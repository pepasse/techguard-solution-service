# TechGuard Solution and Service

Site web professionnel pour TechGuard Solution and Service - Solutions électriques, solaires et d'automation industrielle.

## 📋 Prestations

- **⚡ Installation et Maintenance Électrique** - BTP et industrielle
- **📹 Installation et Maintenance Vidéosurveillance** - Systèmes de sécurité modernes
- **☀️ Systèmes Solaires** - Dimensionnement, installation et maintenance
- **🤖 Programmation Automates** - Siemens, Zélio et autres

## 🛠️ Technologies utilisées

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

## 📁 Structure du projet

```
techguard-solution-service/
├── index.html                 # Page d'accueil
├── css/
│   └── style.css             # Styles personnalisés
├── js/
│   └── script.js             # Scripts interactifs
├── pages/
│   ├── services.html         # Page des services
│   ├── about.html            # Page À propos
│   ├── contact.html          # Page contact
│   └── portfolio.html        # Page portfolio
├── img/                       # Dossier pour les images
└── README.md                  # Ce fichier
```

## 🚀 Démarrage rapide

1. Clonez le repository
2. Ouvrez `index.html` dans votre navigateur
3. Modifiez les informations de contact dans `pages/contact.html`
4. Ajoutez vos images dans le dossier `img/`
5. Personnalisez les couleurs et le contenu selon vos besoins

## 🎨 Personnalisation

### Couleurs
Les couleurs principales sont définies dans `css/style.css` :
- Primary: `#1a1a2e`
- Secondary: `#0f3460`
- Accent: `#ffc107`

### Contact
Mettez à jour les informations de contact dans `pages/contact.html` :
- Téléphone
- Email
- Adresse
- Horaires
